Popper 3

RPG educativo de Física – perspectiva histórica e epistemológica

Sobre o Projeto

Bem-vindo(a) ao Popper 3, um RPG educativo que explora conceitos de Física em um contexto histórico e epistemológico.

Repositório

Código-fonte e documentação no GitHub: github.com/SEU-USUARIO/Popper3

Como Jogar

  1. Clone o repositório:
    git clone https://github.com/SEU-USUARIO/Popper3.git
  2. Instale dependências:
    pip install -r requirements.txt
  3. Execute o jogo:
    python src/main.py
